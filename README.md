# spotify-database-loader
Service that loads spotify's listened tracks stored on Amazon S3 into a PostgreSQL database.

This project is related to another repository called [spotify-recently-played-extractor](https://github.com/GersonLCSJunior/spotify-recently-played-extractor). Please refer to that repository for information, as I'm currently working on writing a more detailed README.

Link to end product: https://spotifyreport.web.app/
